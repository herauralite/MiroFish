Auralite City Layout Reference

This file is a repo-local visual reference asset for the Auralite map shell. Codex should use this as the basis for adapting the city view instead of relying only on placeholder rectangles.

Focus on the overall structure: major road grid, mixed-use blocks, parks, water, industrial areas, civic buildings, and dense urban clustering. Exact reproduction is not required, but the live Auralite map should move closer to this layout language than the current placeholder district blocks.
